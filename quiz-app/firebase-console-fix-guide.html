üîê Step 1: Fix Firestore Security Rules

1.1 Navigate to Firestore Rules

  1. Open Firebase Console - Firestore Rules
  2. You should see the current rules (likely very restrictive)
  3. Click the Rules tab if not already selected

1.2 Replace ALL Firestore Rules

Delete everything in the rules editor and paste this exact code:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Allow authenticated users to read and write their own user documents match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } // Allow authenticated users to read and write their own profile data match /profiles/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } // Allow authenticated users to read and write quiz data match /quizzes/{document=**} { allow read, write: if request.auth != null; } // Allow authenticated users to read and write scores and analytics match /scores/{document=**} { allow read, write: if request.auth != null; } match /analytics/{document=**} { allow read, write: if request.auth != null; } // Allow authenticated users to read and write class management data match /classes/{document=**} { allow read, write: if request.auth != null; } match /classMembers/{document=**} { allow read, write: if request.auth != null; } // Allow authenticated users to read and write settings match /settings/{document=**} { allow read, write: if request.auth != null; } } }
  1. Click the big blue PUBLISH button
  2. Wait for "Rules published successfully" message

2.2 Replace ALL Storage Rules

rules_version = '2'; service firebase.storage { match /b/{bucket}/o { // Allow authenticated users to upload and access their own files match /users/{userId}/{allPaths=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } // Allow authenticated users to access quiz-related files match /quizzes/{allPaths=**} { allow read, write: if request.auth != null; } // Allow authenticated users to access profile pictures match /profiles/{allPaths=**} { allow read, write: if request.auth != null; } // Allow authenticated users to access general uploads match /uploads/{allPaths=**} { allow read, write: if request.auth != null; } } }
